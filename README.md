# MonkshuBasicCrudApp

1. Clone Repository

``` git clone https://github.com/TekmonksJezrelDave/MonkshuBasicCrudApp ```

2. Create Symlink between  Monkshu and Sample
3. Install NPM packages 
   - On /monkshu, run:
  
    ```npm install```
    
    ```npm install mustache sqlite3```

4. Run backend and frontend server
